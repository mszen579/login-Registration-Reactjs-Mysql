You need for this project to:
1- run npm install in the client side
2- run npm install in the server side
3- install and run MAMP
4- create mysql db name:nodejs, username: root, and password: root 
5- import the db from the attached nodejs.sql file in my project
6- run the command: $ nodemon server.js in the server side
7- run the command: $ npm start or yarn start
8- the server is running at port 8000
9- then type in your browser http://localhost:3000
10- you need to know that the validation i have done is on server side for the login and on the client side for the registration
